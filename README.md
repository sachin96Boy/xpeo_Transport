## Expo Transport

This app was designed to use for expo cab service in Elpitiya. its used to get direction on google map and get the distance and time to reach the destination.

## Installation

.flutter pub get
run the app

## technologies

- Flutter
- Dart
- Google Map
- Firebase
- Firebase Auth
- Firebase Cloud Firestore
- geolocator
- google_maps_flutter
